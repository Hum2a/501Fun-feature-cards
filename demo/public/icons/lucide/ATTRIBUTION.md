# Lucide icons (demo only)

Icons in this folder are copied from [Lucide](https://lucide.dev/) via the
`lucide-static` npm package (ISC licence).

Regenerate after changing the catalog:

```sh
npm run icons:sync
```

Curated set: users, hand-heart, party-popper, globe-2, map-pin, target, trophy, star, heart, trending-up, bar-chart-3, sparkles, circle-dollar-sign, zap, beer, smile.
